## Deployment Instructions

### 1. Fork this repository
Click the "Fork" button at the top-right of this page.

### 2. Enable GitHub Pages
1. Go to your repository settings
2. Navigate to "Pages" in the left sidebar
3. Under "Source", select "Deploy from a branch"
4. Select the `main` branch and `/ (root)` folder
5. Click "Save"

### 3. Customize for your domain
To use a custom domain like `https://doksiai.us/google-cloud-showcase`:
1. Add a `CNAME` record in your DNS settings pointing to `[your-username].github.io`
2. In GitHub Pages settings, add your custom domain

### 4. Update Content
Edit the following files to customize:
- `index.html`: Update company information
- `style.css`: Modify colors and styling
- `script.js`: Add custom interactions
